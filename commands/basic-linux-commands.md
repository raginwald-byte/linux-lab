# Basic Linux Commands

This document contains the first Linux commands I learned and practiced during my IT support preparation.

---

## pwd

Displays the current working directory.

Example:

```bash
pwd
```

---

## ls

Lists files and directories in the current location.

Example:

```bash
ls
```

---

## cd

Changes the current directory.

Example:

```bash
cd Desktop
```

---

## mkdir

Creates a new directory.

Example:

```bash
mkdir Linux
```

---

## touch

Creates an empty file.

Example:

```bash
touch Ubuntu.txt
```

---

## echo

Writes text to the terminal or into a file.

Example:

```bash
echo "Hello" > Ubuntu.txt
```

---

## cat

Displays the contents of a file.

Example:

```bash
cat Ubuntu.txt
```

---

## mv

Moves or renames a file.

Example:

```bash
mv Ubuntu.txt Linux/
```

---

## ip a

Displays the network interfaces and IP addresses.

Example:

```bash
ip a
```

---

## ip route

Displays the routing table.

Example:

```bash
ip route
```

---

## ping

Tests network connectivity.

Example:

```bash
ping google.com
```
